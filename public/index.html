How It Works

Three steps to your unified feed

Get up and running in minutes. Configure once, benefit every day.

1

Connect via MCP

Add NeuralPulse to your AI assistant's MCP config. Works with Cursor, Claude Desktop, and any MCP-compatible client out of the box.

2

Add Your Channels

Subscribe to RSS feeds, YouTube channels, or GitHub repos. Tag and organize them however you like - the adapter system handles the rest.

3

Get AI Briefings

Ask your assistant for a briefing. NeuralPulse fetches, filters, and summarizes your content in real time - or delivers it to your inbox on schedule.

Integration

Drop-in configuration

Add NeuralPulse to your MCP config or call the REST API directly.

mcp.json 
{
  "mcpServers": {
    "neuralpulse": {
      "command": "npx",
      "args": ["tsx", "src/index.ts"],
      "env": {
        "ANTHROPIC_API_KEY": "sk-..."
      }
    }
  }
}
REST API 
# Fetch your unified feed
GET /api/feed?limit=20&tags=ai,dev

# Get an AI-generated briefing
GET /api/briefing?since=24h

# List your channels
GET /api/channels

# Health check
GET /health
  -> { "status": "ok" }

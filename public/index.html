Get started for free

Add NeuralPulse to Claude Desktop in under a minute. No account needed.

Set up in Claude Desktop

  1. Open Claude Desktop and go to
    Settings → Developer → Edit Config
  2. Paste this into your claude_desktop_config.json: 
    {
  "mcpServers": {
    "neuralpulse": {
      "url": "https://neuralpulse.manifoldsync.xyz/mcp"
    }
  }
}
  3. Restart Claude Desktop. That's it — ask it "What can NeuralPulse do?" to get started.
